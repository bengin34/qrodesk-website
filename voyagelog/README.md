# VoyageLog marketing site

Static HTML/CSS marketing site for the VoyageLog mobile app.

## Local preview

From the repo root, run:

```bash
python3 -m http.server 8000
```

Then open:

```
http://localhost:8000/voyagelog/
```

You can also open `index.html` directly in a browser for a quick preview.

## Update placeholders

- App Store / Google Play links in `index.html`
- Contact email (`support@yourdomain.com`) across pages
- Legal placeholders in `privacy.html`, `terms.html`, and `impressum.html`
- Open Graph image (replace `assets/og-image.svg` if desired)
- Screenshot placeholders in `assets/placeholder-screenshot-*.svg`
