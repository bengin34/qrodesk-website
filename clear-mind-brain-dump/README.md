# clearMind marketing site

Static HTML/CSS marketing site for the clearMind brain dump app.

## Local preview

From the repo root, run:

```bash
python3 -m http.server 8000
```

Then open:

```
http://localhost:8000/clear-mind-brain-dump/
```

You can also open `index.html` directly in a browser for a quick preview.

## Update placeholders

- App Store / Google Play links in `index.html` (if needed)
- Contact email (`support@qrodesk.com`) across pages
- Legal placeholders in `terms.html` (governing law)
- Open Graph image in `assets/og-image.svg` if branding changes
